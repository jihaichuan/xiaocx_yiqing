# 小熊猫登记小助手

项目简介
面对疫情，我们小程序公司开发了一款针对疫情登记小程序软件，希望各企业、线下酒店，超市，小区等管理者，可以对出入人员进行登记，欢迎使用。

## 特性

1. 扫码登记；
2. 统计数据；
3. 每日数据统计汇总，发送邮件到指定邮箱；
4. 打印出入登记码，A4纸直接打印；

## 依赖
无
